# Nigeria PAYE Tax Calculator

A web-based utility for calculating Nigerian income tax (PAYE) according to current tax laws and regulations.

## Features
- **Accurate Calculations**: Computes PAYE based on the latest Finance Act.
- **Consolidated Relief Allowance**: Automatically calculates CRA and taxable income.
- **Easy to Use**: Simple HTML/JS interface for quick assessments.
- **Privacy Focused**: All calculations are performed locally in the browser.
